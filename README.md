# Analyzing-and-Visualizing-Regional-Sales-Performance-
sales_performance_Analysis project

Sales Performance Analysis Project – Summary

Successfully analyzed sales data across multiple regions (East, West, North, and South) to evaluate business performance.
Performed data cleaning and preprocessing to correct inconsistencies in region names, product categories, and missing values.
Used Excel functions, Pivot Tables, and Pivot Charts to transform raw data into meaningful insights.
Created an interactive dashboard to visualize key performance indicators (KPIs) such as Sales, Profit, Quantity Sold, and Discount.
